# 📡 Telecommunications Laboratory Experiments (11–20)

This repository contains laboratory experiments performed using the **Emona Telecoms-Trainer 101**.
The experiments cover key topics in **analog and digital communication systems**, including modulation, digital encoding, spread spectrum communication, and software-defined radio concepts.

Each experiment demonstrates practical implementation of communication principles through hardware modules and oscilloscope observation.

# Overview

Modern telecommunications systems rely on both **analog and digital signal processing techniques**.
These laboratory experiments introduce students to the practical implementation of communication systems such as:

* Analog Modulation
* Digital Modulation
* Pulse Code Modulation (PCM)
* Spread Spectrum Communication
* Bandwidth Limiting
* Software Defined Radio (SDR)

Using the **Emona Telecoms Trainer 101**, these experiments allow students to observe signal generation, modulation, transmission, and recovery using real hardware modules.

# Experiment List

## Experiment 11 – Amplitude Modulation (AM)

This experiment introduces the concept of **Amplitude Modulation**, where the amplitude of a carrier signal varies according to the message signal. Students observe the modulated waveform and analyze its frequency components.

---

## Experiment 12 – AM Demodulation

This experiment focuses on recovering the original message signal from an AM signal using demodulation techniques such as envelope detection.

---

## Experiment 13 – Double Sideband Suppressed Carrier (DSBSC)

In this experiment, a **DSBSC signal** is generated using a multiplier module. The carrier is suppressed, leaving only the sidebands.

---

## Experiment 14 – DSBSC Demodulation

The original message signal is recovered using a **product detector**, demonstrating synchronous detection.

---

## Experiment 15 – Frequency Modulation (FM)

This experiment demonstrates **frequency modulation**, where the carrier frequency changes according to the message signal.

---

## Experiment 16 – Phase Modulation (PM)

Phase modulation is studied by observing how the phase of a carrier signal changes relative to the input signal.

---

## Experiment 17 – Pulse Code Modulation (PCM)

PCM converts analog signals into digital form through **sampling, quantization, and encoding**.

---

## Experiment 18 – Bandwidth Limiting in PCM Systems

This experiment investigates how **bandwidth limitations affect digital signals**, causing distortion and signal degradation.

---

## Experiment 19 – Direct Sequence Spread Spectrum (DSSS)

DSSS spreads a signal across a wide bandwidth using a **pseudo-noise (PN) sequence**, improving resistance to interference and jamming.

---

## Experiment 20 – Undersampling in Software Defined Radio (SDR)

This experiment demonstrates how **undersampling techniques** can recover signals even when the sampling frequency is lower than twice the carrier frequency, provided the signal bandwidth is limited.

# Learning Outcomes

After completing these experiments, students should be able to:

* Understand the principles of **analog and digital modulation**.
* Analyze communication signals using an **oscilloscope**.
* Implement **modulation and demodulation circuits**.
* Understand **digital signal encoding techniques** such as PCM.
* Investigate the effects of **bandwidth limitations on communication systems**.
* Understand **spread spectrum communication techniques**.
* Explore **software-defined radio concepts** such as undersampling.
